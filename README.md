# AR Navigation
-제물포 역을 기준으로 지도를 생성하여
유저가 제물포 주변에 있는 시설들을 잘 이용 할 수 있도록 AR Navigation, PhotoZone 등, 다양한 정보를 제공하는 Android Application입니다.
AR기술을 활용하여 => 네비게이션 사용시 실시간으로 바닥에 화살표를 띄워 유저에게 직관적으로 길을 알려주고
포토존이나 특정 랜드마크에 가면 3D 오브젝트와 촬영을 하거나,해당 랜드마크의 설명을 위해 3D 캐릭터가 등장해 설명을 해주는 등각종 상호작용도 할 수 있도록 기획 되어 있습니다
